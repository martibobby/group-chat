About the Project!

Description

This web application is designed to allow multiple users to participate in a chatroom through video, audio, or text. Each user is able to enter their name and choose what color will represent them in the chat. They can also enter a specific room if they wish. The project is build with webRTC and socket.io and using jQuery. The styling and layout is primarily created with Bootstrap v4.0.
